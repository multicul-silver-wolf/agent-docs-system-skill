# UX Principles for User-Loved Products

1. Time-to-first-value beats feature depth.
2. Every screen must answer: what can user do next?
3. Remove mandatory fields until absolutely needed.
4. Empty states should teach the next action.
5. Errors must be actionable, never vague.
6. One page, one primary CTA.
7. Copy clarity > cleverness.
8. Trust signals near conversion points (proof, guarantees, examples).
