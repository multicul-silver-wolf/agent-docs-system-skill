# Discover Checklist

- [ ] ICP is specific (role, context, budget level)
- [ ] Problem is acute/chronic, not vague preference
- [ ] User currently uses workaround
- [ ] Frequency is at least weekly
- [ ] Existing alternatives listed with pros/cons
- [ ] Clear why-now trigger exists
- [ ] Kill criteria defined

## Scoring (quick)
- Pain intensity (1-5)
- Frequency (1-5)
- Willingness to pay/time (1-5)

Proceed if total >= 10.
